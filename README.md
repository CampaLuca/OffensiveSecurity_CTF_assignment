# OffensiveSecurity_CTF_assignment

## Compile
```sh
gcc rot_key.c -o rot_key
```

## Running

### Replicating the behaviour of the given executable
```sh
./rot_key 
```
or
```sh
./rot_key authenticate
```


### Printing the secret code to use with your PIC
```sh
./rot_key crack
```
